# Amazon Brazil SQL Analytics Project

## Business Intelligence Case Study using SQL Server

**Prepared By:** RANJEET ANADA

---

## Project Overview

This project analyzes the Amazon Brazil E-Commerce dataset using SQL Server to solve real-world business problems. The analysis focuses on customer behavior, seller performance, product categories, payments, reviews, and order delivery. The objective is to transform raw business data into meaningful insights that support data-driven decision-making.

---

## Project Objectives

* Analyze customer purchasing behavior.
* Evaluate seller performance across different regions.
* Identify high-performing product categories.
* Analyze payment methods and customer reviews.
* Measure order delivery performance.
* Generate business insights using SQL.

---

## Dataset Information

This project is based on the Amazon Brazil E-Commerce dataset and includes **11 relational tables**.

* Orders
* Order Items
* Order Payments
* Order Reviews
* Products
* Sellers
* Geolocation
* Leads Qualified
* Leads Closed
* Product Category Translation
